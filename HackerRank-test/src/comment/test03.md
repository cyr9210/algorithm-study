# test 03

### 문제
- 게임 설정
- 상황 갯수 입력
- +값 입력 : 체력 증가
- -값 입력 : 체력 감소
- 체력은 항상 1이상
- **최소한의 기본 체력값을 구하시오.**

### 샘플 테스트 케이스
10   
-5   
4   
-2   
3   
1   
-1   
-6   
-1   
0   
5   
 
### 샘플 결과
8
 
### 해설
[나의 코드보기](https://github.com/cyr9210/algorithm-study/blob/master/HackerRank-test/src/test3.java)
- 초기 기본 체력값을 1로 둔다.
- 상황에 해당하는 값을 하나씩 더하며 판단한다.
    - 값이 음수가 됬을 때, 체력이 1이상이 되도록 한다.
        - 값이 음수가 됬을 때, 그 음수값 만큼 기본 체력값에 더해준다.
